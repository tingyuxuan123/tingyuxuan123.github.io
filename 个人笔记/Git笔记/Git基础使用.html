Git基础使用

路远 2023-03-11 git

# git 配置

  1. git config --global user.name " "
  2. git config --global user.email " "
  3. 检查信息是否写入成功
    1. git config --list

# git 的基本操作

  1. git init 初始化
  2. git add
  3. git commit
命令 说明
git init 初始化仓库
git clone 拷贝一份远程仓库，也就是下载一个项目。
命令 说明
git add 添加文件到仓库
git status 查看仓库当前的状态，显示有变更的文件。
git diff 比较文件的不同，即暂存区和工作区的差异。
git commit 提交暂存区到本地仓库。
git reset 回退版本。
git rm 删除工作区文件。
git mv 移动或重命名工作区文件。
git remote add origin git@github.com:tingyuxuan123/search-ht.git
git branch -M main
git push -u origin main

# git 仓库的创建

  1. git init
  2. git add . 或一个文件名 // .表示添加全部文件
  3. git commit -m ”描述“
  4. git remote add origin 远程仓库地址
  5. git branch -M main //创建分支
  6. get push origin master

# github 提交

  1. git add .
  2. git commit -m " 描述 "
  3. git push
